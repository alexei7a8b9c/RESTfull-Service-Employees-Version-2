# Employee Management System

RESTfull сервис для управления сотрудниками с поддержкой ролевой модели доступа.

## Технологии
- Java 21
- Spring Boot 3.2.0
- PostgreSQL
- Flyway
- MapStruct
- Lombok
- Spring Security
- Swagger/OpenAPI 3

## Запуск приложения

### Предварительные требования
- Docker и Docker Compose
- Java 21
- Maven 3.6+

### Быстрый запуск с Docker

1. Клонируйте репозиторий
2. Запустите базу данных:
```bash
docker-compose up -d
